## Project Overview
This project focuses on classifying customer credit risk into
Low, Medium, and High categories using machine learning.

## Dataset
The dataset includes customer demographic, financial, and
loan-related features such as income, loans, savings, employment status,
and credit score.

## Model Used
- Random Forest Classifier

## Evaluation Metrics
- Confusion Matrix
- Accuracy
- Precision
- Recall
- F1-score

- Bivariate analysis (box plots)
- Multivariate analysis (correlation heatmap)
